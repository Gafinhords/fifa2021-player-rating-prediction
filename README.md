# FIFA 21 Player Rating Prediction

## Задача
Предсказать рейтинг (`overall`) футболиста на основе его игровых характеристик.

## Данные
- Источник: Kaggle (FIFA 21 Complete Player Data)
- Размер: ~18k игроков
- Признаки: возраст, позиция, команда, национальность, количество ударов (`hits`), потенциальный рейтинг (`potential`)

## Подход
1. **EDA** — анализ зависимостей, визуализация
2. **Feature engineering** — target encoding для категориальных признаков (команда, национальность)
3. **Модель** — RandomForestRegressor
4. **Оценка** — RMSE = 1.38 (ошибка ~1.4 балла рейтинга)

## Ключевые выводы
- Потенциальный рейтинг (`potential`) исключён из модели, так как это не наблюдаемый параметр в реальных условиях
- Национальность и команда влияют на рейтинг через target encoding
- Возраст оказался важнее количества ударов

## Бизнес-ценность
Модель может использоваться скаутской службой для первичной оценки игроков на основе статистики, без привязки к их рыночной стоимости или медийному потенциалу. Ошибка в 1.4 балла рейтинга означает, что модель ошибается не более чем на один "уровень качества" (например, между 80 и 81).
